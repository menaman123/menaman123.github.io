Real-time Environmental Monitoring with RFID

System Architecture

Hardware Setup

  • Inpoint PIPtag multi-sensors
  • Linux VM with libusb drivers
  • Custom C data parsers

Software Stack

  • Node.js server with Express
  • Socket.io real-time updates
  • Nodemailer alert system

Key Functionalities

  • Mean-filtered sensor data (5-sample buffer)
  • Threshold-based email alerts (temperature/humidity/light)
  • Auto-calibration of sensor readings
  • Web dashboard with real-time visualization
