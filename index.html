Research Projects

Parallel Computing Benchmark: Spark vs MPI on AWS

Comparative analysis of gravity simulation performance between traditional pySpark implementation and cloud-optimized MPI on AWS EC2 infrastructure.

Key Contributions:

  • Developed O(n²) particle trajectory simulator with both Python/Spark and C/MPI implementations
  • Achieved 196x speedup using MPI on AWS t2.micro instances
  • Implemented automatic scaling with EC2 placement groups
Case Study Full Report Presentation

IoT Environmental Monitoring with RFID

Real-time ambient data collection system using PIPtag RFID sensors with automated email alerts and mean-filtered data visualization.

Technical Features:

  • Linux VM data pipeline with libusb C drivers
  • Node.js server with socket.io real-time updates
  • Nodemailer threshold alerts with Gmail OAuth2
System Demo Technical White Paper

Optimal Control Through RL Policy Iterations

Advanced research in reinforcement learning applications for dynamic systems control and game theory.

Key Studies:

Zero-Sum Differential Games
  • Compared Lyapunov vs Riccati iterations (25 vs 45 steps convergence)
  • Implemented algebraic Riccati solver in MATLAB
  • Analyzed pursuit-evasion and cybersecurity applications
Caltech Ducted Fan Control
  • Vaisbord-Milhstein RL algorithm implementation
  • 18-iteration convergence to LQ optimal control
  • 6DOF system stabilization with Q=I₆, R=I₂
Game Theory Paper Control Systems Full Dissertation
